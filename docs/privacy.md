# Política de Privacidade — Tarefas Diárias

Última atualização: 21 de fevereiro de 2026

## 1. Introdução

A extensão **Tarefas Diárias** foi desenvolvida com foco em simplicidade e privacidade.

## 2. Coleta de Dados

A extensão **não coleta, armazena ou transmite dados pessoais para servidores externos**.

Não são coletados:
- Nome
- E-mail
- Endereço IP
- Localização
- Dados financeiros
- Histórico de navegação
- Conteúdo de sites visitados

## 3. Armazenamento Local

As tarefas criadas pelo usuário (nome, data e tempo) são armazenadas exclusivamente no navegador por meio do `localStorage`.

Esses dados:
- Permanecem apenas no dispositivo do usuário
- Não são enviados para servidores
- Não são compartilhados com terceiros

## 4. Compartilhamento

Nenhum dado é vendido ou transferido.

## 5. Remoção de Dados

O usuário pode apagar os dados usando o botão "Limpar tudo" ou removendo a extensão.
